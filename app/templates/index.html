Deterministic PDF thumbnails

Upload a PDF or image; get stable URLs keyed by SHA-256 of the file. Deploy on Vercel with optional Blob storage.

# Register (multipart field: file)
curl -sS -X POST "https://your-app.vercel.app/api/v1/documents/" \
  -F "file=@./document.pdf" | jq .

# Thumbnail (1-based page index)
curl -o thumb.png \
  "https://your-app.vercel.app/api/v1/documents/<document_id>/pages/1.png"

Deterministic URLs

document_id is SHA-256 of the file. Paths /api/v1/documents/{id}/pages/{n}.png never change for the same content. Set BLOB_READ_WRITE_TOKEN on Vercel for durable storage across instances.
